About This Project

This project focuses on improving digital accessibility for users with cognitive conditions, particularly autism and ADHD. It explores inclusive web design through simulations, data visualisations, and interactive tools based on user feed back and web development guidelines.

Personal Motivation

The motivation for this work stems from both academic interest and personal experience. With a brother who has Autism and ADHD and a sister with Autism, I’ve seen firsthand the challenges that digital platforms can present. This project aims to raise awareness and design better, more accessible digital experiences that are inclusive for neurodivergent users.
